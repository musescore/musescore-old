MuseScore ist ein freies, GNU GPL lizensiertes grafisches Notensatzprogramm für Linux und andere Betriebssysteme. Einige Eigenschaften:
  • grafische Seitendarstellung, Noten werden auf einem "virtuellen Notenblatt" eingegeben
  • beliebige Anzahl von Notenzeilen
  • Mehrstimmig
  • einfache und schnelle Noteneingabe mit Maus, Tastatur oder Midikeyboard
  • integrierter Sequencer und FluidSynth Softwaresynthesizer
  • Import und Export im MusicXml- und Midifileformat.
  • Plattformunabhängiger Code für Linux, Windows und Mac
  • GNU GPL Lizenz
  • ...
