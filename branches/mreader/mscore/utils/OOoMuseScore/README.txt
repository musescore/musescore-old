Very simple test-version! By no way a "release" version!

TESTED ON WINDOWS ONLY!
Modifications need to be done in Subs.xlb -> Function CallLilyPond() 
to make it work in LINUX.

Requirements:

- OpenOffice 2
- MuseScore 0.9
- ImageMagick
- GhostScript (for pdf to png conversion)


Installation:

- Check ooolilypond.sourceforge.net for installation info.
- Edit the settings in Config dialog

These settings work for me:

Template:
default

Template Path:
I:\WebDevelop\OooMuseScore\OOoLilyPond_080716\templates\

MuseScore Executable:
C:\Program\MuseScore 0.9\bin\mscore.exe

ImageMagick Executable:
C:\Program\ImageMagick-6.3.8-Q16

*******************************************************

Please note that any modifications you make to the code
are saved in the OpenOffice [User]/Application Data/OpenOffice.org2/user/basic 
directory!
